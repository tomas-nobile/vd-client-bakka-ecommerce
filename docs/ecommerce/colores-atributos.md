# Colores disponibles para atributos

El tema reconoce automáticamente estos nombres como colores. Podés usarlos como valores del atributo "Color" en WooCommerce y el punto de color en la card y en la ficha de producto se muestra solo.

Funciona con mayúsculas, minúsculas o mixto: "Negro", "NEGRO" y "negro" dan el mismo resultado.

Para colores que no están en la lista, usá el código hex directamente como valor: `#4db6ac`.

## Usar un color personalizado

Si el color que necesitás no está en la lista, poné el código hex directamente como valor del atributo:

1. En Productos → Atributos → Configurar términos
2. Nombre del término: `Verde Agua` (lo que ve el cliente)
3. Slug: `#00CED1` (el hex — el tema lo detecta automáticamente)

O al revés: nombre `#00CED1`, el tema lo usa como color y muestra el hex en el tooltip.
