# Spec: Home Blog — Cards tipo Instagram (Posts)

## Resumen

Rediseñar la sección Blog del front-page para que las cards parezcan post de Instagram. Las cards muestran multimedia (imágenes/videos extraídas de bloques Gallery y Paragraph del post), descripción corta (una oración + "..."), sin título. Al hacer clic se abre un modal con multimedia grande y descripción completa. Overlay oscuro, clic en overlay cierra el modal. Animación inside out sutil en cards y modal, manteniendo la animación fade-up existente.

## Datos desde WordPress

### Multimedia

- **Imágenes**: bloques **Gallery** y/o **Paragraph** del post (`$post->post_content`).
  - Gallery: `parse_blocks( $post->post_content )`, buscar `core/gallery`, leer `$block['attrs']['ids']` (array de attachment IDs).
  - Paragraph: si contiene imagen embebida, revisar `core/image` en `innerBlocks` o `innerHTML`.
  - Fallback: imagen destacada (`get_post_thumbnail_id( $post->ID )`).
- **Video**: si existe bloque `core/video` o `core/embed` (YouTube, Vimeo), extraer `src`/`url` desde `attrs`.
- **Orden multimedia**: [imágenes de Gallery] + [imagen destacada si no está en Gallery] + [video si existe].

### Descripción

- **Card**: primera oración del excerpt o contenido; truncar a ~12 palabras + "...". Función sugerida: `wp_trim_words( get_the_excerpt( $post ) ?: wp_trim_words( $post->post_content, 40 ), 12 ) . '...'`
- **Modal**: excerpt completo o contenido (sin truncar).

## Diseño de la card

Inspirado en `0design-decoypisos/app/core/components/Reviews/ReviewsCard.tsx`:

- **Estructura**: bloque multimedia arriba, descripción debajo, fecha estilo Instagram.
- **Sin título** (como en Instagram).
- **Multimedia**:
  - Una imagen: mostrar sola, `object-cover`, altura máxima ~350–400px.
  - Varias imágenes: carrusel con flechas anterior/siguiente (estilo Instagram). Inline SVG igual que `image-modal.php`.
  - Video: reproductor `<video>` o embed; mismo área que las imágenes.
- **Fecha**: estilo y ubicación similar a Instagram (pequeña, arriba o esquina; ej. `position: absolute; top: 8px; left: 8px`).
- **Descripción**: una oración + "..." debajo de la media.
- **Clic en la card**: abre modal (no enlace al post).

---

## Modal

- **Trigger**: clic en la card.
- **Contenido**: multimedia más grande (desktop tamaño destacado; móvil adaptado) + descripción completa debajo.
- **Overlay**: fondo oscuro (`bg-black/80` o `bg-black/90`), `fixed inset-0`, `z-index` alto.
- **Cerrar**: clic en overlay o botón X; opcional tecla Escape.
- **Patrón**: igual que `src/single-product/components/image-modal.php` (backdrop `data-close-modal`, contenido centrado).
- **Navegación**: si hay varias imágenes en el post, flechas prev/next dentro del modal.

---

## Animaciones

1. **Fade-up**: mantener `data-aos="fade-up"` en cabecera, grid y botón (como en `blog.php` actual).
2. **Inside out (sutil)**:
   - Cards: entrada con `opacity: 0 → 1`, `scale: 0.95 → 1`, origen centrado. Duración ~0.3–0.4s.
   - Modal: al abrir `scale: 0.95 → 1` + fade; al cerrar inverso.
   - Usar transitions CSS, easing suave (e.g. `ease-out`).

---

## Textos

- **Cabecera**: "Noticias y Artículos" / "Nuestras Últimas Entradas" (o equivalentes i18n).
- **Botón**: **"Ver más posts"** (no "artículos") — enlace a la página de posts/blog.
- **Accesibilidad**: `aria-label` en flechas y botón cerrar; `role="dialog"`, `aria-modal="true"` en modal.

## Pseudocódigo helper multimedia

```php
function etheme_get_post_multimedia( $post ) {
    $items = [];
    $blocks = parse_blocks( $post->post_content );
    foreach ( $blocks as $block ) {
        if ( $block['blockName'] === 'core/gallery' && ! empty( $block['attrs']['ids'] ) ) {
            foreach ( $block['attrs']['ids'] as $id ) {
                $items[] = [ 'type' => 'image', 'id' => $id ];
            }
        }
        if ( $block['blockName'] === 'core/video' && ! empty( $block['attrs']['src'] ) ) {
            $items[] = [ 'type' => 'video', 'url' => $block['attrs']['src'] ];
        }
    }
    if ( empty( $items ) ) {
        $thumb = get_post_thumbnail_id( $post->ID );
        if ( $thumb ) {
            $items[] = [ 'type' => 'image', 'id' => $thumb ];
        }
    }
    return $items;
}
```
