# Leer el spec antes de codear

Cada feature tiene su spec en `specs/`. Leerlo antes de tocar código.

## Mapa de specs

| Spec | Feature |
|---|---|
| `1.home.md` | Home page |
| `2–7.home-*.md` | Secciones del home |
| `8.blog.md` | Blog / posteos |
| `9–11.archive-product-*.md` | Listado de productos |
| `12.single-product.md` | Detalle de producto |
| `13.page-cart.md` | Carrito |
| `14.page-checkout.md` | Checkout |
| `15.page.md` | Mi cuenta (login / registro / panel) |
| `16.contact.md` | Contacto |
| `17.navbar.md` | Navbar |
| `18.security.md` | Capa de seguridad JS |
| `19.information-pages.md` | Páginas legales |
| `20.footer.md` | Footer |

## Regla

Si no existe un spec para lo que se pide, solicitarle al usuario que lo defina antes de implementar.
